# Rose Lab v8｜Instagram 編輯風格

本版重點：

- 使用奶油白、玫瑰粉、森林綠作為統一主色
- 全站字體優先使用「Microsoft YaHei UI／微軟雅黑 UI」
- 保留頁首 MENU、透明 Logo、搜尋與手機選單
- 首頁分類改為簡潔方塊，不使用大量裝飾
- 最新文章改成雙欄編輯式卡片，手機自動單欄
- 文章頁增加留白、柔和重點框、圓角表格與清楚目錄
- 表格在手機上可左右滑動
- 頁尾維持簡單文字與必要連結

## 更新方式

將此資料夾內所有檔案上傳至原 GitHub repository 根目錄，覆蓋同名檔案即可。請先備份你已修改的：

- `src/content/blog/`
- `public/` 中自行新增的圖片

Cloudflare 會在 GitHub 更新後自動重新部署。
